# Outreach templates — by opportunity type

These are STARTING POINTS. The skill MUST customize each draft with specifics from the target page or query. Templates that go out unmodified read as templates and don't work.

All drafts use Bill's voice as defined in config.yaml — direct, specific, no consultant-speak.

---

## Template 1: Source-request response (HARO / Featured / Qwoted)

**Subject:** [match the query — e.g., "AI implementation timeline question"]

**Body:**
```
Hi [Journalist first name],

[1-2 sentence direct answer to the query, with a specific number or example.]

[1-2 sentence supporting context from real experience. Cite a pattern, not a vague generalization.]

[Optional: 1 sentence with a contrarian/non-obvious point that makes the quote interesting.]

If useful, you can attribute to:
Bill Colbert, Founder, Treetop Growth Strategy (treetopgrowthstrategy.com)

Happy to provide more detail if helpful. My direct line: [phone] or bill@treetopgrowthstrategy.com.

Bill
```

**Customization required:** the answer must be specific to the question. "AI implementation usually takes 6-12 weeks for mid-market" beats "AI implementation timelines vary."

## Template 6: Community comment (Reddit / Indie Hackers / etc.)

**Format:** 80-200 words, conversational, no marketing language.

**Structure:**
```
[1-2 sentences acknowledging the OP's specific situation/question. Not generic.]

[3-5 sentences of actual useful insight — share what you've seen work or fail, with specifics. The substance must hold up if removed from any link.]

[Optional 1 sentence: "I wrote about this here if helpful: [URL]" — only if the link genuinely answers a deeper version of the OP's question.]

[Sign-off if appropriate to the platform — Reddit doesn't need one; Indie Hackers / Pavilion benefits from "— Bill (founder of Treetop, fwiw)"]
```

**Hard rule:** if the comment without the link wouldn't be valuable, don't post it. The link must be incidental, not the point.

## Across all templates: tone calibration

Bill's voice is:
- Direct, not breezy
- Specific over generic
- Acknowledges effort/imposition ("small ask," "no urgency")
- Self-aware about being self-interested without being apologetic
- Never uses: "thrilled," "pleased," "industry-leading," "cutting-edge," "revolutionize," "synergies," "thought leader"

**Test:** read the draft out loud. If it sounds like a marketing email, rewrite. If it sounds like Bill talking to a peer, ship it.
